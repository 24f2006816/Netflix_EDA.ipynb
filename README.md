---

# 📌 Project Overview

Netflix has become one of the world's largest streaming platforms, offering thousands of Movies and TV Shows across multiple countries and genres.

This project analyzes the Netflix dataset to answer business-oriented questions through data analysis and visualization. Using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn, the dataset is cleaned, transformed, and explored to identify trends in content production, ratings, genres, countries, and release years.

The primary goal is to demonstrate practical Data Analysis skills while extracting valuable insights from real-world data.

---

# 🎯 Objectives

- Perform data cleaning and preprocessing
- Handle missing values and duplicate records
- Analyze the distribution of Movies and TV Shows
- Explore Netflix content across countries
- Analyze release year trends
- Identify the most common genres
- Examine content ratings
- Discover the most active directors
- Identify frequently appearing actors
- Generate business insights through visualization

# 📂 Dataset

- **Dataset:** Netflix Movies and TV Shows
- **Source:** Kaggle
- **Creator:** Shivam Bansal
- **Dataset Link:** https://www.kaggle.com/datasets/shivamb/netflix-shows
- **File Used:** `netflix_titles.csv`

### Dataset Features

The dataset contains information about Netflix Movies and TV Shows, including:

- Show ID
- Type (Movie / TV Show)
- Title
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre (Listed In)
- Description

> **Note:** The dataset is **not included** in this repository due to Kaggle's redistribution policy. Please download it manually from Kaggle and place `netflix_titles.csv` inside the project directory before running the notebook.

# 📊 Key Insights

- 🎥 Movies account for a larger share of Netflix's content than TV Shows.
- 🌍 The United States is the largest contributor to Netflix's library, followed by India.
- 📈 Netflix experienced rapid content growth between **2017 and 2019**.
- 🎭 Drama, International Movies, and Documentaries are among the most common genres.
- ⭐ TV-MA is the most frequently assigned content rating.
- 🎬 Recent years have seen a significant increase in Netflix Originals.

# 🚀 How to Run

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Netflix-Data-Analysis.git
```

### Move into the project folder

```bash
cd Netflix-Data-Analysis
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Download Dataset

Download **`netflix_titles.csv`** from:

https://www.kaggle.com/datasets/shivamb/netflix-shows

Place the dataset inside the project folder.

### Run the Notebook

Open the notebook using:

```bash
jupyter notebook
```

or
